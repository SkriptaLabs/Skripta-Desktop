## Ideen 

Skripta ist eine Desktop App für das wissenschaftliche Schreiben mit KI. KI wird nicht als Gegenspieler gesehen, sondern als Mitstreiter, der Produktivät und professionelles Arbeiten stärkt. 

Skripta bietet einen MCP Server an, durch den Agents über wohl definierte Schnittstellen, dem Schreiber zuarbeiten können. So soll eine saubere Aufgabenteilung ermöglicht werden. 

## File Struktur und Code-Organisation

app/feature
---> components (Visuelle Komponenten)
---> service (Business Logik)
---> data-access (Datenzugriff)
---> MCP (MCP Schnittstellen)
